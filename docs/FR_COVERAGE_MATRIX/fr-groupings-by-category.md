- **Scope:** CLI entry, county API, listing extraction, image download, maps, schools, data preservation, provenance
- **Epic Coverage:** E2 (7 FRs) + E1 (1 FR)
- **Priority:** All P0
- **Status:** Ready for implementation

### Kill-Switch & Filtering (FR9-14, FR49)
- **Scope:** HARD/SOFT criteria, verdict evaluation, failure explanations, configuration
- **Epic Coverage:** E3 (6 FRs) + partial E4
- **Priority:** All P0
- **Status:** Ready for implementation

### Scoring & Ranking (FR15-20, FR25, FR27, FR48)
